## ① 노름 분리 판정 — **불합격 → 순수 비정규화 레시피 확정** (객관 규칙 자체 적용)

| | 기준 | 노름 분리 (contrast_proj) | 순수 비정규화 (bridge) |
|---|---|---|---|
| t2a top-1 | ≥ 60 | **9.3% ✗** | 59.3 |
| zero-shot | ≥ 55 (std≤2) | (미측정 — t2a에서 조기 탈락) | 52.1±0.8 |

투영 헤드가 InfoNCE 그래디언트를 흡수해 g 본체에 언어 접지가 형성되지 않음
(SimCLR에서 투영 후 표현이 더 좋은 것과 동일 기제 — 우리는 g "본체"를 쓰므로 역효과).
**최종 레시피 = joint + 비정규화 + hybrid λ0.3 (contrast_head 없음)** — 이후 전 신규 런 적용.

## ② DINOv2-DZ 폐루프 — **65.2% [60.9–69.2]: 예측 [87,93] 대실패** (H2 역전 신호 1)

| | 오프라인 (1차 표) | 폐루프 |
|---|---|---|
| DINOv2-DZ | **dec 0.740 / a2z 56.2% (전 지표 1위)** | **65.2%** (SR@220 63.4) |
| CLIP-DZ (동일 프로토콜) | dec 0.684 / a2z 35.1% | **87.0%** |

- 태스크별 [98,46,48,96,76,42,92,46,88,20] — 분산 극심 (98 vs 20). 실패: grasp 123,
  wrong_object 17, reach 7 (CLIP 계열은 wo/reach ≈ 0 — **물체 혼동·도달 실패가 신규 등장**).
- phase2 오프라인은 건강 (MAE 0.08, lat_cos 0.608 — CLIP 계열보다 높음) → 학습 실패 아님.
- **해석 (오프라인-폐루프 역전, proprio에 이은 2번째 사례)**: DINOv2 임베딩은 Δz-액션
  코딩이 정밀하나(월드 상태 민감), 그만큼 **정책 관측으로서는 분포 이탈에 취약**
  (뷰포인트·조명·자기폐색의 순간 변화에 z_t가 크게 움직임 → receding horizon 오차 증폭).
  CLIP의 언어 정렬 학습이 부여하는 의미 불변성(semantic invariance)이 폐루프 강건성의
  실체일 가능성 — **H2 판정에 결정적 재료** (G2 기준: 언어 정렬 SR ≥ DINOv2 −10%p →
  실측 +21.8pp로 초과 충족).
- ledger의 RADIO 발동 조건("코딩=DINO, 언어=SigLIP2 교착")은 **불발동** — 교착이 아니라
  CLIP 계열이 폐루프에서 이기는 구도로 정리되는 중. SigLIP2-HY03 폐루프가 최종 확인.
