## 판독 (실행자 소견 — G5 판정은 폐루프 후)

1. **recon 경로는 정렬 방식에 강건**: R² 0.67~0.68로 사실상 동률 — recon·cycle 유지
   규약이 contrastive 붕괴를 막았고, 정렬 타깃 교체가 복원 능력을 훼손하지 않음.
2. **명확한 트레이드오프 (사전 등록 예상과 일치)**: Δz 정렬(a→Δz retrieval)은
   DZ 30.2% ≫ HY01 16.3% ≫ HY03 4.4% ≫ DA 0% 순으로, 언어 정렬(t2a)은 정확히 역순.
   λ_c가 두 정렬의 연속 보간 다이얼로 동작함이 확인됨.
3. **핵심 발견 — 언어 접지는 g-공간에 형성되나 h-디코딩 경로로 전이되지 않음**:
   DA/HY의 t2a 검색은 강하지만(59~62%), E_text를 h에 직접 넣는 zero-shot 디코딩은
   전 팔에서 우연 수준. DZ가 낮을 것이라는 사전 등록 예상은 맞았으나 DA도 낮음.
   해석: h는 (실제 Δz, g(a)) 입력 분포에서 학습되고, InfoNCE는 방향(cos)만 정렬하므로
   E_text는 h 입력 매니폴드의 오프-분포 점 (스케일 재조정으로 미해소 — 두 스케일 모두 측정).
   → zero-shot 언어→액션을 원한다면 (i) 검색 기반 디코딩(문장→최근접 g→h), 또는
   (ii) 텍스트 임베딩을 h 학습 분포에 포함(hybrid의 cycle을 text에도) 필요. 분석자 판단 요청.
4. HY03이 t2a 최고(62.1%)이면서 R² 손실 최소(−0.003) — hybrid 관점에서 λ=0.3이
   0.1보다 유리해 보이나, a→Δz 정렬 급락(4.4%)이 폐루프에 미칠 영향이 미지수.
   **G5 판정에 필요한 폐루프 50롤아웃이 다음 단계.**
