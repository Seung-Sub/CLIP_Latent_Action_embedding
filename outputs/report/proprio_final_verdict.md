판정 규칙(분석자, 사전 등록): suite SR ≥ 83.8 (DZ 기준 CI 하한) → 채택 검토 / 둘 다
미달 → (c) 기각 + 부정 결과 문서화. **추가 변형 임의 시도 금지 준수 — 본 문서로 종결.**

## 진단 종합 (부정 결과의 내용)

1. **드롭아웃 절반 회복, 완치 실패**: 59.0→73.8 (+14.8pp) — 지름길 의존을 줄이나
   기준까지 13.2pp 부족. shortcut 완화가 부분적으로만 유효하다는 정량 데이터.
2. **그리퍼 2D도 유해 (−19.4pp)**: 인과 혼동의 잔여 채널 — 그리퍼 qpos는 자신의 직전
   그리퍼 명령의 에코(액션 피드백)로, 파지 실패 시 "열려 있음" 상태가 정책을 오도.
   wrong_object 18건(전 조건 최다)이 시각 대신 상태 관성을 따른 흔적과 정합.
3. **wrong_goal 실패의 지속** (44~61건): proprio 계열 공통 — 파지 후 배치 단계에서
   시각 검증 없이 궤적 관성으로 진행하는 패턴.
4. 문헌 정합: causal confusion (de Haan '19), BC에서 proprio/history 유해 사례
   (RT-1 절제, Mandlekar '21 등). 우리 기여: **D4(정보량)·P6(오프라인 예측력)·폐루프
   3축 분리 측정**으로 "정보 있음 ≠ 사용 가능"을 단일 통제 셋에서 정량화.
