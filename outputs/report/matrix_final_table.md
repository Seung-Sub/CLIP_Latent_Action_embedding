## 판독 (실행자)

1. **H2 방향 정리**: 순수 Δz 코딩(오프라인)은 DINOv2 압도, 폐루프는 CLIP 계열 압도
   (+21.8pp) — "언어 정렬 표현의 의미 불변성 = 폐루프 강건성" 서사. P7은 이탈도 가설을
   기각했으므로 기제 규명은 "과제 관련 부분공간 안정성" 후속 필요.
   **서베이 문서(최종보고서_v2)의 최대 리스크 항목("문헌은 dynamics에 CLIP 회피")에 대한
   정면 반증 실험이 완성됨** — 논문 방어의 중심 결과.
2. **SigLIP2-HY03 80.2 vs CLIP-HY03-unnorm 81.0**: 동일 레시피(비정규화) 기준 0.8pp 차이
   — <3pp 반복 규칙 대상이나, 시드 상이(2 vs 1) + vocab 편향 미해소 상태라 반복 런보다
   **v3 재판정 후 재비교가 우선** (분석자 판단 요청).
3. **예측 장부**: DINOv2 ❌, SigLIP2 폐루프 ❌ (3연속 폐루프 예측 실패) — 개정 방법론
   (코딩 축 + 강건성 축 분리, P7 재정의 포함)의 필요성이 데이터로 누적됨.
4. **대표 수치 동결 보류 권고 유지**: s1 81.0 vs s2-정규화 87.0의 레시피×시드 교란 —
   절제 1런(비정규화@seed2) 승인 대기.
